# 🚀 Coding Upsolves

Welcome to **Coding Upsolves**!

This repository is a collaborative collection of accepted solutions and upsolves from competitive programming contests. Our goal is to strengthen our problem-solving skills by participating in contests, upsolving unsolved problems, and maintaining a well-organized archive of solutions.

---

# 🎯 Objectives

* Participate in weekly and biweekly programming contests.
* Upsolve unsolved problems after every contest.
* Learn new algorithms and data structures.
* Practice clean coding and teamwork.
* Maintain a repository of high-quality accepted solutions.

# 📌 Repository Rules

* One accepted solution per problem.
* Organize solutions according to the contest.
* Use meaningful commit messages.
* Pull the latest changes before pushing.
* Test all solutions before committing.
* Keep code clean, readable, and well-structured.
* Respect other contributors' work and avoid overwriting solutions without discussion.
